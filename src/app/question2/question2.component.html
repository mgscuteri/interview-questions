RxJS Reactive FizzBuzz

This is a reactive take on the popular "fizzbuzz" question. Below, you will see three values, one of which is user input, with the other two being random numbers generated at two second intervals. The challenge here is to update the "fizzBuzzResult" observable such that it dynamically reacts and displays the updated fizz buzz result whenever any of the three values change.

Based on the divisor values and the user input, the fizzbuzz result should:

  • Display "fizz" if the user input value is evenly divisible by the "Fizz Divisor"
  • Display "buzz" if the user input value is evenly divisible by the "Buzz Divisor"
  • Display "fizzbuzz" if the user input value is evenly divisible by the "Buzz Divisor"
